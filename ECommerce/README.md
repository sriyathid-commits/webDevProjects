# 🛍️ e-commerce website (Mini Flipkart Clone)

A beginner‑friendly e‑commerce web application built with **React, Node.js, Express, and MongoDB**.  
This project demonstrates product listing, search, category filtering, cart management, and checkout flow.

---

## 🚀 Features
- 📦 Product listing with images, description, and price
- 🔎 Search bar to filter products by name
- 🗂️ Category filter (Electronics, Accessories, etc.)
- 🛒 Cart with quantity update and remove‑item functionality
- 🧾 Checkout page with product details and purchase confirmation
- 🌱 MongoDB seeding for sample products

---

## 🛠️ Tech Stack
- **Frontend:** React, Axios, React Router  
- **Backend:** Node.js, Express, Mongoose  
- **Database:** MongoDB  
- **Version Control:** Git & GitHub  

---

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/sriyathid-commits/webDevProjects.git
cd webDevProjects
2. Backend setup
cd ECommerce/server
npm install
node seed.js   # seed sample products
node server.js # start backend on port 5000
3. Frontend setup
cd ../client
npm install
npm start      # start frontend on port 3000
