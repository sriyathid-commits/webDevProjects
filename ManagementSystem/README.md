# 🧾 Management System

A robust management system built with **React (frontend)** and **Node.js/Express + MongoDB (backend)**.  
It supports CRUD operations for employees, departments, and tasks, with role‑based access control and analytics dashboards.

---

## 🚀 Features
- Employee & Department CRUD operations
- Role‑based access control (Admin/User)
- Analytics dashboard with charts
- Exportable reports (CSV/PDF)
- Responsive UI with React

---

## 🛠 Tech Stack
- **Frontend:** React, React Router, Axios
- **Backend:** Node.js, Express.js, MongoDB
- **Authentication:** JWT
- **Styling:** CSS / Bootstrap
