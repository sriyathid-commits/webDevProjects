# 💬 Chat Application

A real‑time chat app built with **React (frontend)** and **Node.js/Express + Socket.IO (backend)**.  
Supports private and group messaging, online/offline indicators, and persistent message history.

---

## 🚀 Features
- Real‑time messaging with Socket.IO
- Multiple users chatting simultaneously
- Online/offline user indicators
- Simple and responsive UI
- Easy to extend with usernames & timestamps

---

## 🛠 Tech Stack
- **Frontend:** React, Axios, Socket.IO Client
- **Backend:** Node.js, Express.js, Socket.IO
- **Styling:** CSS
